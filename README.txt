Easy way to add custom post types without having to recreate all the boilerplate meta code.

== Description ==


== Installation ==

1.Upload:
Unzip RCPT.zip file, and upload all files to your wordpress folder: <root>/wp-content/plugins/RCPT

2.Activate:
In the admin panel, find Plugins->Plugins,click it, active RCPT.

3.Use:

You will need to edit the custom_post_types.php file at the top with arrays describing your custom post types.
The admin panel us still under development.
